Проект написан на языке C++ в Visual Studio 2022(Стандарт ISO C++17 (/std:c++17)). 
На вход программа принимает один файл “input.csv” в котором хранится матрица совместности и массив весов. 
В результате работы программа заполняет файл “output.csv” матрицей пяти лучших гипотез, соответствующими им весами и временем работы программы.
